## 🔥 Hauptmerkmale (Key Features)

* **👷 Worker-Interface:** Ultra-einfache Zeiterfassung optimiert für Smartphones auf der Baustelle. Arbeiter wählen Datumsbereiche und loggen Arbeitszeit, Pausen und Baustellen mit 1 Klick.
* **👨‍🍳 Chef-Bereich (Admin Panel):** Erweiterte Datenanalyse. Filtern Sie Arbeitsstunden nach Mitarbeitern, Datumsbereichen oder Baustellen mit automatischer Echtzeit-Berechnung.
* **📄 Gesetzeskonformer PDF-Export:** Generiert saubere, offizielle "Arbeitszeitnachweis"-Berichte in Schwarz-Weiß mit Unterschriftenfeldern für Arbeitgeber und Arbeitnehmer, bereit für die Buchhaltung.
* **🔒 Cyber-Security:** Sitzungsisolation, sichere JWT-Token, einwegverschlüsselte Bcrypt-Passwörter und ein integrierter automatischer Passwortgenerator.
* **🐳 Modern Tech Stack:** Python FastAPI, PostgreSQL (Dockerized) und Vue 3 (Composition API).
